# AgentDraw Layout Systems

Layout systems describe how information is arranged on the canvas. They are separate from design
styles.

- A layout system controls composition, reading path, focal point, proportions, alignment, and
  information priority.
- A design style controls palette, typography, stroke, geometry, component treatment, and mood.

Choose the provider first, then choose the layout system, then choose the design style.

## Rule

Do not use layouts as rigid templates. Use them as composition systems: preserve the defining
reading path, alignment discipline, focal area, and spacing rules, while adapting proportions and
content density to the user's source.
